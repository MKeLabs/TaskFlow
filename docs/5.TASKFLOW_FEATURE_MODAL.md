# Step 5: Modal Component Feature

This step covers the reusable modal component used for create/edit dialogs.

## What & Why

This step introduces a reusable modal dialog component. Modals are used for forms and confirmations, like creating or editing projects. Making a generic modal keeps your code DRY and your UI consistent across the app.

## 1. Modal Component TypeScript (`src/app/shared/ui/modal/modal.component.ts`)
```typescript
import { Component, EventEmitter, Input, Output } from '@angular/core';
import { CommonModule } from '@angular/common';

@Component({
  selector: 'app-modal',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './modal.component.html',
  styleUrl: './modal.component.scss'
})
export class ModalComponent {
  @Input() title = '';
  @Input() description = '';
  @Input() saveLabel = 'Save';
  @Input() cancelLabel = 'Cancel';
  @Input() busy = false;

  @Output() cancel = new EventEmitter<void>();
  @Output() confirm = new EventEmitter<void>();

  onBackdropClick() {
    if (!this.busy) {
      this.cancel.emit();
    }
  }

  onCancelClick() {
    if (!this.busy) {
      this.cancel.emit();
    }
  }

  onConfirmClick() {
    if (!this.busy) {
      this.confirm.emit();
    }
  }
}
```

## 2. Modal Component HTML (`src/app/shared/ui/modal/modal.component.html`)
```html
<div class="modal-backdrop" (click)="onBackdropClick()">
  <div class="modal" (click)="$event.stopPropagation()">
    <h2>{{ title }}</h2>
    @if (description) {
      <p class="muted">{{ description }}</p>
    }

    <ng-content />

    <div class="modal-actions">
      <button class="secondary" (click)="onCancelClick()" [disabled]="busy">{{ cancelLabel }}</button>
      <button (click)="onConfirmClick()" [disabled]="busy">{{ saveLabel }}</button>
    </div>
  </div>
</div>
```
